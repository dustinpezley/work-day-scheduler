# Work Day Scheduler

## Description
Simple web-based app to plan your day. Current date is shown at the top and your plans should carry over day over day using localStorage.

 - Individual rows per hour from 9AM to 5PM
 - Save button sets items in localStorage
 - Rows are shaded based on current time of day.
